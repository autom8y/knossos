# Strategy Document Templates

> Templates for strategic analysis and planning artifacts.

## Purpose

Provides structured templates for strategic planning: initiative prioritization with OKRs, competitive intelligence gathering, market sizing (TAM/SAM/SOM), and scenario-based financial modeling.

## Template Catalog

| Template | Purpose | Agent |
|----------|---------|-------|
| [Strategic Roadmap](templates/strategic-roadmap.md) | Initiative prioritization, resource allocation, OKRs | strategy-lead |
| [Competitive Intel](templates/competitive-intel.md) | Competitor profiling, product analysis, monitoring plan | market-analyst |
| [Market Analysis](templates/market-analysis.md) | TAM/SAM/SOM sizing, segment analysis, growth drivers | market-analyst |
| [Financial Model](templates/financial-model.md) | Scenario projections, unit economics, sensitivity analysis | finance-analyst |

## When to Use Each Template

| Scenario | Template |
|----------|----------|
| Quarterly/annual planning | Strategic Roadmap |
| Evaluating competitive threats | Competitive Intel |
| Sizing a new market opportunity | Market Analysis |
| Modeling financial impact of a decision | Financial Model |
| Setting OKRs | Strategic Roadmap |
| Preparing board materials | Financial Model + Market Analysis |

## Quality Gates Summary

| Template | Gate Criteria |
|----------|---------------|
| **Strategic Roadmap** | Initiatives scored, resources sum to capacity, OKRs measurable |
| **Competitive Intel** | Consistent ratings, predicted moves reasoned, sources documented |
| **Market Analysis** | TAM/SAM/SOM with methodology, segments sized with growth rates |
| **Financial Model** | Three scenarios, unit economics benchmarked, assumptions stated |
