# Aesthetic Evaluation

> Research-grounded frameworks for evaluating interface visual quality and aesthetic perception.

## Overview

Aesthetic judgments form in 50ms and remain stable across longer exposures. Perceived aesthetics correlates strongly with perceived usability (r=0.79, d=0.86) -- the aesthetic-usability halo is both an opportunity and a risk. Computational metrics explain up to 49% of variance in aesthetic ratings; the remaining 51% requires human or agent judgment informed by validated frameworks.
