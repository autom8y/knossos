# CLI Reference: help

> Get help for any `ari` command.

`ari help` provides help text for any command in the application.

**Family**: help
**Commands**: 1

---

## Synopsis

```bash
ari help [command] [flags]
```

## Examples

```bash
# Help for the root command
ari help

# Help for a specific command
ari help session
ari help session create

# Equivalent short form
ari session --help
ari session create --help
```

For full option details, run `ari help --help`.
