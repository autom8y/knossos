# CLI Reference: version

> Display binary version, build metadata, and platform info.

`ari version` prints the build version, commit hash, build date, Go version, and platform architecture.

**Family**: version
**Commands**: 1

---

## Synopsis

```bash
ari version [flags]
```

## Examples

```bash
# Human-readable output
ari version

# Machine-readable JSON output
ari version -o json
```

For full option details, run `ari version --help`.
