A metadata block embedded in managed files (like CLAUDE.md) using `<!-- KNOSSOS:START tag -->` and `<!-- KNOSSOS:END tag -->` markers. Inscriptions allow the sync pipeline to update managed sections idempotently while preserving user-authored content outside the markers.
